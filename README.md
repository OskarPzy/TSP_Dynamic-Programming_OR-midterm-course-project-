## TSP solved using Dynamic Programming 
Solve TSP using Dynamic Programming in python, for Operations Research course midterm project

### Intro
本文文字内容为中文，所解问题是10个点的TSP问题，10个点选为中国10个大城市，通过经纬坐标粗略计算各点之间的直线距离
